# Security Policy

## Responsible Use
CyberSentinel Pro is designed for blue-team SOC learning, log analysis, and defensive security reporting. Use it only on logs and systems you own or have permission to analyse.

## Secrets
Do not commit `.env`, API keys, ngrok tokens, exported reports containing sensitive evidence, or private log files.

## Reporting Issues
If you find a security issue in this project, open a GitHub issue without sharing real credentials, internal IPs, private logs, or customer data.
